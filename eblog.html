Eblog blogging website

The website is a online blogging website which is completely responsive. This is a fullstack project in PERN Stack

Users can create an account , login , add blogs , search blogs by keywords, save blogs , edit owned blogs , edit profile etc

Technologies:
  • - Javascript
  • - NodeJS + ExpressJS
  • - React with Redux
  • - Material UI
  • - Postgresql
