E-Commerce Website

This was a project I started back in march 2021 when I first learned nodejs.

The website is a fully functional e commerce website where users can register or login and buy products.

Users are shown the cart and users can change products from cart then order them.

Users are given both online payment functionality and also cash on delivery method.

Admin can login to admin panel and change/add/delete products,ship an existing order

Technologies:
  • - Javascript
  • - NodeJS + ExpressJS
  • - hbs template
  • - Razorpay payment gateway
  • - MongoDB
